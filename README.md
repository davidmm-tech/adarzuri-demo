# adarzuri-demo

Demo en Next.js para captación y diagnóstico inicial con IA para Adarzuri.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Producción

El proyecto compila en modo producción con `npm run build` y no depende de fuentes externas (Google Fonts), por lo que funciona correctamente en entornos restringidos.
